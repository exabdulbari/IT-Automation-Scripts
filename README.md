# IT Automation Scripts
Simple PowerShell scripts to automate common IT tasks for troubleshooting and maintenance.

## Scripts
- DiskCheck.ps1: Checks free disk space.
- NetworkPingReport.ps1: Pings a list of devices and reports status.
- TempCleaner.ps1: Cleans temporary files.

## Usage
1. Open PowerShell as Administrator.
2. Run the script, e.g. `.\DiskCheck.ps1`.
3. Review output in console.
